# JCCycleView
---
JCCycleView是一个可以循环展示View的控件，类似于scrollView的分页效果，并提供`增加`，`删除`，`替换`View的方法

## 安装
---
### CocoaPods

`pod 'JCCycleView'`

### 手动安装
将`JCCycleView`文件夹中的所有代码拖拽到项目中


## 使用方法
---
请参考`JCCycleView.h`文件


## 系统要求
---
该项目最低支持`iOS 7.0`和`Xcode 7.0`
